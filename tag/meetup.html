Another entry in what is apparently an uninterrupted string of Novapy event updates!

Last night @gekitsuu spoke about Rackspace and Python and Zombies. He covered Python integration with libcloud and MongoDB and how to work that into a Rackspace deployment. We all learned a little bit and had a lot of new faces in the crowd as well.

I announced the Dgentle Django class this August as well. The class sold out before midnight! It will be an amazing time I’m sure. We are finalizing the last details now, and will be updating everyone along the way.
